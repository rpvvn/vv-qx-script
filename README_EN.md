## 📖 Introduction
This repository is used to store personal small scripts developed for daily use, including but not limited to:
- Scripts for intercepting and extracting specific parameters (e.g., QQ Farm code extraction)
- Scripts adapted to Quantumult X (Circle X) rewrite mechanism
- Small utility scripts for mobile terminal efficiency improvement
- Other personalized custom scripts

All scripts are developed for personal learning and use, and are continuously updated and optimized.

### 📋 Script List
| Script Name | Function Description | Adapted Tools |
|-------------|----------------------|---------------|
| ✔ QQ-farm-code | Automatically intercept and extract the code parameter from QQ Farm request URL, copy it to clipboard, and block the request to avoid code invalidation | Quantumult X (Circle X) |
| ✔ QQ-farm-AutoUpdateCode | Automatically obtain QQ_Farm credentials and submit them to the server for updates | ProxyPin |
| ✔ Extensible Script 2 | (Supplement according to actual scripts) | (e.g.: Quantumult X/Tampermonkey/Shortcuts) |


## 📦 Usage
1. Clone or download the script file you need
2. Configure according to the script's corresponding documentation (see script comments for details)
3. Adapt to tools such as Quantumult X (Circle X) and enable relevant rules
